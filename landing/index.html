Developer Tools

Meet bgit.

A CLI for developers who juggle multiple Git accounts. Set up identities once, switch between them safely, and keep SSH plus repo ownership rules under control.

✓
Switch between work, personal, and client identities with one command
✓
Workspaces for automatic identity selection by folder
✓
Automatic SSH key and configuration management for each identity
✓
Built-in diagnostics with `bgit doctor` and repair helpers
✓
Repo binding and normal `git` command compatibility after setup
Install bgit → Read Docs
bgit — zsh
~ bgit setup ⚡ bgit setup complete Managed SSH config and pre-push checks installed
~ bgit add ✓ Added identity: work Name, email, GitHub username, and SSH key saved
~/projects/work-app bgit use work ✓ Switched to identity: work Updated user.name, user.email, and SSH key selection
~/projects/work-app bgit status → Effective identity: work Source: workspace binding · Remote aligned
~/projects/work-app bgit doctor --fix ✓ SSH permissions verified and config checks passed
